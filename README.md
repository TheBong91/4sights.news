# 4sights.news SEBA Master Web Application

Main components
- Angular 4
- Angular-Material 2
- Angular CLI (Typescript, ...)
- express
- mongoose (MongoDB Client)
- Authentication with JWT

Reference Point
- angular2-full-stack boilerplate

Install
``` bash
$ npm Install
```

Seed Database with predefined `categories`
``` bash
$ npm run db:seed
```
